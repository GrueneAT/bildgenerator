## Out of Scope

| Feature | Reason |
|---------|--------|
| Sub-Kategorien pro Bundesland | Unnötige Komplexität, Suche funktioniert flach |
| Bundesland-Metadata-Datei | LLMs brauchen kein Mapping für zukünftige Einträge |
| Eigene Logo-Bilddateien pro Organisation | Text-Overlay-System funktioniert |
| UI-Code-Änderungen | Nur Datenänderung nötig |
| Änderungen an logo_json.py | Script wird für diese Aufgabe nicht benötigt |
